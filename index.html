Dicionário de HTML

Tag de vídeo

Tag de vídeo permite colocar na tela um vídeo local ou da internet.

Tag de template

Armazena algum conteúdo que ficará oculto quando a página for carregada

Tag de span

Usado para colorir uma parte de um texto

Tag de picture

Oferece aos desenvolvedores web mais flexibilidade na especificação de recursos de imagem

Tag de meta

Descreve metadados em um documento HTML.

Tag de link

Link para uma folha de estilo externa

Tag de form

Um formulário HTML com dois campos de entrada e um botão de envio

Tag de div

A tag

define uma divisão ou seção em um documento HTML.

Tag de canvas

É usada para desenhar gráficos dinamicamente, por meio de script (geralmente JavaScript).

Tag de blockquote

Especifica uma seção citada de outra fonte.

Tag de map

É usado para definir um mapa de imagem. Um mapa de imagem é uma imagem com áreas clicáveis.

Tag de img

É usado para incorporar uma imagem em uma página HTML

Tag de dialog

Define uma caixa de diálogo ou subjanela

Tag de body

Define o corpo do documento

Tag de track

tag especifica faixas de texto para elementos áudio ou vídeo.
